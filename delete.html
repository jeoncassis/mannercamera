데이터 삭제 요청

앱명: 매너카메라  |  개발자: 전병훈

매너카메라 앱과 관련된 데이터 삭제를 요청하실 수 있습니다. 아래 안내에 따라 직접 삭제하거나 개발자에게 문의해 주세요.

1. 기기 내 데이터 직접 삭제

매너카메라는 개발자 서버에 데이터를 저장하지 않습니다. 기기에 저장된 데이터는 다음과 같이 직접 삭제할 수 있습니다.

촬영한 사진·동영상 삭제 기기의 갤러리 앱 → DCIM/MannerCamera 폴더에서 직접 삭제
앱 설정 데이터 삭제 기기 설정 → 앱 → 매너카메라 → 저장공간 → 데이터 삭제
앱 완전 제거 앱을 삭제하면 앱과 관련된 모든 설정 데이터가 기기에서 삭제됩니다.

2. 광고 식별자(AD ID) 삭제 또는 재설정

Google AdMob이 사용하는 광고 식별자는 기기 설정에서 직접 관리할 수 있습니다.

  • 기기 설정 → Google → 광고 → 광고 ID 삭제 (Android 12 이상)
  • 기기 설정 → Google → 광고 → 광고 ID 재설정 (Android 11 이하)
  • 또는 맞춤 광고 수신 거부를 선택하여 추적을 제한할 수 있습니다.
